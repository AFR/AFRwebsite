Nos projets

L'équipe AFR Software, développe différents projets web, d'applications web et mobiles dont voici quelques exemples.

Centre CrossFit Minos

Le Centre CrossFit Minos a fait appel à AFR Software pour développer son site web ainsi que sont application de réservation en ligne.

Les membres du gym peuvent voir en temps réelles les places restantes à un entraînement et s'inscrire afin de réserver leur place.

Le site web est piloté par un CMS Wordpress et l'application a été développée en Google Web Toolkit hébergée sur les serveurs de Google App Engine.

Site web Centre CrossFit Minos
Camping Ruisseau Bleu

Le Camping Ruisseau a fait appel à AFR Software pour réorganiser son image sur le web.

Les site montre les différents services offerts au camping, les évènnements, une galerie de photos et les coordonnées du camping.

Le site web est piloté par un CMS Wordpress et l'équipe AFR s'est assuré que le client puisse mettre à jour aisément les informations de la page.

Site web Camping Ruisseau Bleu
Yarrrrr!

Le jeu Yarrrrr! réalisé dans notre cours de programmation de jeu-vidéo, met en scène un pirate trouvant une boussole lui permettant d'inverser la gravité.

Il a remporté la bourse Frima Studio car il a été jugé comme étant le plus ambitieux et accompli, tout en démontrant une mécanique de jeu innovante, simple et bien réalisée.

Yarrrrr! a été réalisé avec le moteur de jeu Unity3D.

Technologies utilisées

Wordpress

Outil de gestion de contenu de site web afin de façilement garder votre page web à jour en y ajoutant ou modifiant son contenu.

Site officiel Wordpress

Google Web Toolkit

Google Web Toolkit (GWT) est un outil de développement pour la construction et l'optimisation de complexes applications basées sur navigateur. GWT est utilisé par de nombreux produits de Google, il est open source et utilisé par des milliers de développeurs à travers le monde.

Site officiel GWT

Google App Engine

Grâce à Google App Engine, vos applications Web peuvent profiter des mêmes systèmes que ceux utilisés par les applications Google elles-mêmes. Développement rapide, déploiement accéléré, administration simplifiée, évolutivité sans effort, etc.

Site officiel GAE

NodeJS

NodeJS est une plateforme javascript permettant le développement d'application réseau légères et rapide en temps réel

Site officiel Node
